Setup Required

This booking page needs to be connected to a Google Sheet via a Google Apps Script Web App URL.

1 Create a Google Sheet and add your time slots.
2 Open the Apps Script editor and paste the provided Code.gs.
3 Deploy as a Web App and copy the URL.
4 Share the booking link in the format below:
https://yourwebsite.com/?api=YOUR_APPS_SCRIPT_URL

See the included README.md for detailed instructions.
