Setup Required

This booking page needs to be connected to a Google Sheet via a Google Apps Script Web App URL.

Step 1: Create a Google Sheet

  1. Go to Google Sheets and create a new spreadsheet
  2. Create separate tabs for each date/service (e.g., "June 10", "Tuesday", "Services", etc.)
  3. In each tab, set up the columns:
    • Column A: Time (e.g., "09:00 AM", "10:00 AM", etc.)
    • Column B: Status (set to "open" for available slots, "close" for blocked times)
    • Column C+: Booking fields (e.g., Name, Phone, Email, etc.)
  4. Example:
    Time Status Name Phone Email
    09:00 AM open
    10:00 AM close

Step 2: Deploy Google Apps Script

  1. In your Google Sheet, go to Tools → Apps Script
  2. You'll see Code.gs file. Delete the default content and paste this code:
    Code.gs
    Loading Code.gs...
  3. Save the script (Ctrl+S)

Step 3: Deploy as Web App

  1. Click Deploy in the top right
  2. Select New Deployment
  3. In the "Select type" dropdown, choose Web app
  4. Set the deployment settings:
    • Execute as: Me
    • Who has access: Anyone
  5. Click Deploy
  6. You'll see a popup with your webapp URL (looks like: https://script.google.com/ macros/d/YOUR_SCRIPT_ID/exec)
  7. Copy the Web App URL (you'll need this in the next step)

Step 4: Create Your Booking Link

  1. Paste your Apps Script URL from Step 3 into the link below:
    2. https://eberjohns.github.io/booking/?api=
  2. Your booking link:
    https://eberjohns.github.io/booking/?api=YOUR_APPS_SCRIPT_URL
  3. Copy and share this link with your customers!
💡 Tip: You can set the Status column to "close" to block time slots during breaks or lunch times. The booking system will not allow bookings on "close" slots.
